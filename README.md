# Image Converter

A tool to convert 128x64px PNG images to a bitmap array for use on a 128x64 OLED display

## Description

This script uses the `sharp` library to convert an image to a 128x64 bitmap array in C for use with an SSD1306 screen in an Arduino project.

## Installation

1. Clone the repository or download the script.
2. Navigate to the project directory.
3. Install the required dependencies using npm:

```sh
npm install
```

## Usage

1. Place the image you want to convert in the root folder of the project and name it image.png.
2. Run the script using Node.js:

```sh
node converImageToBitmap.js
```
3. The script will generate a bitmap.h file containing the bitmap array.
